Cheat Sheet in case you forget

Published: Fri 03 December 2010
By Jason Jabbour

In Review.

  • This is a quick cheat sheet just in case you forget how to make things work.
  • When you create a new .md page, you need to back out of the content folder into the root folder and use the "make html" command. This will convert your markdown files into a HTML file so you can view it.
  • View your site on your https://localhost:8000 by using the "make server" command.

  • Both html and starting the server can by done in one command by concantanating them -> make html && make serve
